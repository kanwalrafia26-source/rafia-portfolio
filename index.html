About me

Building software that stays out of people's way.

I'm a final-year Software Engineering student at NUML, Islamabad, with a strong foundation in software engineering principles and a habit of writing clean, structured, maintainable code from day one. I've built everything from REST APIs to full-stack mobile applications, and I genuinely enjoy solving the small problems that make software actually pleasant to use.

My main craft is Flutter and Firebase mobile development — I like turning a real problem into an app people actually want to open twice. Lately I've been pairing that with Python and early AI work, exploring how AI can enhance user experiences rather than just existing for its own sake.

Whether it's a reminder app or a therapy tool, I care about the same thing: does this genuinely make someone's day easier.
